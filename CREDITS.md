# Credits and licenses

## Foundation

This project began from [Kenney's Starter Kit FPS](https://github.com/KenneyNL/Starter-Kit-FPS), used under the MIT license. The starter kit's included 2D sprites, 3D models, fonts, and sound effects are CC0. Its original license is retained as `LICENSE.md`.

The game-specific bank-heist systems, procedural environment, UI, scoring, inventory, escalation, AI variants, mission flow, and Windows tooling were created for this project.

## Bundled files

- `fonts/`, `models/`, `sounds/`, `sprites/`, and `vector/`: Kenney Starter Kit FPS. Assets CC0.
- The original starter scripts/scenes remain in `objects/`, the top-level legacy `scripts/` files, and `weapons/` as a reference foundation. The game runs from `scenes/main.tscn` and the organized subfolders under `scripts/`.

## Character art and animation

- [Quaternius Ultimate Modular Men Pack](https://quaternius.com/packs/ultimatemodularcharacters.html) — CC0. The bundled master FBX supplies security, police, SWAT, and civilian male outfits, faces, skeleton, pistol, and 24 animations.
- [Quaternius Ultimate Modular Women Pack](https://quaternius.com/packs/ultimatemodularwomen.html) — CC0. The bundled master FBX supplies female outfit variants, faces, skeleton, attached pistol, and the matching animation set.
- [Quaternius Universal Animation Library](https://quaternius.com/packs/universalanimationlibrary.html) — CC0. The Godot Standard GLB is retained as an additional animation source for future expansion.

The source license text is retained under `assets/quaternius/modular_characters/License.txt` and `assets/quaternius/animation_library/License.txt`.

## Generated wall artwork

- `assets/art/wall_art_collection.png`: original four-panel wall-art texture generated for this project with OpenAI's built-in image-generation tool. It contains a desert mesa, historic bank facade, blue/gold architectural study, and Southwestern highway image. No external copyrighted source asset was supplied.
- `assets/art/museum_art_collection.png`: original four-panel museum-art texture generated for this project with OpenAI's built-in image-generation tool. It contains an ancient Mediterranean maritime scene, a Renaissance-inspired astronomical study, a Southwestern ceremonial landscape, and a botanical specimen. No external copyrighted source asset was supplied.
